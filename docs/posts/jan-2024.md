大家好，新年快乐！在告别2023年之际，让我们花点时间回顾一下FMHY今年的非凡历程：

- 我们制作了一个新网站，丰富了各种精心设计的功能（当然是由我制作的），以提升您的体验，
  并使在所有设备上（包括移动设备）浏览维基变得愉快！
- 我们开设了Discord服务器并达到了10,000名成员，这为社区引入了许多优秀的新成员，
  也使贡献变得更加容易
- 我们在reddit上已达到25万+成员和3100万次浏览量！
- 我们的FMHYedit仓库见证了社区做出的令人印象深刻的1000+个拉取请求，
  并合并了超过8000次提交！
- 这个维基已经发展成为不仅仅是盗版索引，所以我们觉得是时候让其他部分也得到关注。
  因此，我们将工具部分组织并分离到自己的页面中，这使我们能够将许多有用的工具从存储中移出，
  让它们更容易被发现
- 我们开始了FMHY的精简计划，链接数量从27,000减少到了24,000！我们删除了许多无效、无用和小众的
  链接/类别，并重新组织了许多部分以获得更清晰的界面和更简单的导航

进入2024年，我们热切期待在我们的成就基础上继续建设并共同成长。没有这个社区每个人的爱与支持，
FMHY就不会有今天的成就。感谢您成为FMHY的重要组成部分。祝新的一年里继续成长和成功！💙

### 维基更新

- 为 **[SearX](https://searx.fmhy.net/)** 和 **[Whoogle](https://whoogle.fmhy.net/)**
  创建了我们自己的实例。

- 在我们的网站上创建了一个 [Posts](/posts) 页面，因此您无需访问reddit即可查看所有更新。
  您也可以通过 [RSS](/feed.rss) 关注。

- 创建了新的 [Social Media Tools](/social-media-tools) 类别，并为需要它们的热门网站
  添加了子部分。（Reddit、Discord、YouTube等）

- 添加了 [Game Soundtracks](/audio#game-soundtracks)、
  [IRC Tools](/downloading#irc-tools) 和
  [Coding AIs](/developer-tools#coding-ais) 部分。

- 将职业部分拆分为子部分：[Remote Jobs](/misc#remote-jobs)、
  [Tech Jobs](/misc#tech-jobs)、[Startup](/misc#startup) 和
  [Finance / Crypto](/misc#finance-crypto)

- 将游戏模组部分拆分为子部分：
  [Mod Indexes](/gaming#mod-indexes) 和
  [Single Mods](/gaming#game-mods)

- 将新闻部分拆分为子部分：[Aggregators](/misc#aggregators) 和
  [Tech News](/misc#tech-news)

- 重新组织了
  [Android Operating Systems](/mobile#operating-systems) 和
  [App Launchers](/mobile#app-launchers)，并将它们从存储中移出。

### 新增推荐 ⭐

- 在流媒体网站中推荐了 [Braflix](/video#streaming-sites)。快速流媒体，
  无广告，界面美观，支持4K + 1080p。

- 在阅读网站中推荐了 [Liber3](/reading#ebooks)。庞大的图书馆，
  界面美观，一键下载。

- 在视频播放器中推荐了 [Screenbox](/video-tools#video-players)。开源，
  基于VLC的视频播放器，界面美观。

- 在Discord客户端中推荐了 [Vesktop](/social-media-tools#discord-clients)。
  轻量级桌面客户端，使用Vencord构建。

- 在AI聊天机器人中推荐了 [SydneyQt](/ai#proprietary-llms)。越狱版Bing AI。

- 在FOSS APK中推荐了 [App Manager](/mobile#foss-apks)。开源
  Android应用程序包管理器。

- 在Android工具中推荐了 [scrcpy](/mobile#android-device)。通过桌面管理
  移动设备。

- 在Android设备中推荐了 [KernelSU](/mobile#android-device)。一些人比Magisk
  更喜欢的Root工具。

- 在iOS隐私中推荐了 [Onion Browser](/mobile#ios-privacy)。
  Tor在其网站上推荐。

- 在自制软件中推荐了 [Wii Guide](/gaming#homebrew)。

- 在视频工具中推荐了 [Video2x](/video-tools#video-tools-1)。无损
  视频/GIF/图像放大器。

- 在Windows更新中推荐了 [VCRedist](/system-tools#windows-updates)。轻松
  下载所有Microsoft Visual C++运行库。

- 在隐私扩展中推荐了 [CanvasBlocker](/privacy#privacy-extensions)。
  防止Javascript API指纹识别。

### 已移除项目

- 移除了SevenGamers，因为它们
  [决定关闭](https://i.imgur.com/C5sgVqI.png)。

- 移除了YasDL，因为它们上传了带有与filecr和其他已被移除的网站相同恶意软件的StarDock版本。

- 移除了Torrentz2，因为它们已停止索引新种子。

- 取消推荐了LookMovie，因为它们在视频播放前添加了
  [10秒等待时间](https://i.imgur.com/I0D9Hyt.png)。

- 取消推荐了游戏工具中的DS4Windows。您可以直接将游戏添加到Steam中，
  然后右键单击游戏启用PS4控制器支持。

- 取消推荐了Ideogram，因为它不再无限使用。

- 移除了EveryNoiseAtOnce，因为开发者已被
  [Spotify解雇](https://i.imgur.com/AaIrcAc.png)。

- 移除了CryptoStorm，因为它[可能不安全](https://i.imgur.com/VDlSY1T.png)。
  所有者显然已与其脱离关系，但很难信任一个历史如此黑暗的公司。

- 从NotABug移除了我们的备份。他们的网站此时几乎
  [每天都会宕机](https://i.imgur.com/Vx8Ou68.png)，在那保持25个维基页面更新已成为噩梦。
  如果他们解决了这些问题，我们可能会恢复。
