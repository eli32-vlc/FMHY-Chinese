# 维基更新

- 添加了FMHY的**[Alternative Frontend](https://ffmhy.pages.dev/)**完全不同的设计. 这来自官方来源,所以它将与新的编辑保持同步. 它还有一个每页的 [random site](https://i.ibb.co/fVkHqhRP/image.png) / [2](https://i.imgur.com/p4Mxs8y.png)按. 注意这项目的首页已经被删除,现在它直接转向维基 itself.感谢 nw为这项制作.

- 加入了**[3 New Instances](https://fmhy.net/other/backups)**到我们的备份页面.

- 作为我们颜色选择器的选项.谢谢Samidy为这件事.

- 增加了新的节目为 [Textbooks](https://fmhy.net/reading#textbooks).

- 增加了新的节目为 [File Info / Metadata](https://fmhy.net/file-tools#file-info-metadata).

- 合并Linux+Mac Torrent客户端到 [Main Section](https://fmhy.net/torrenting#torrent-clients),并添加标签+平台图标. [Before vs After](https://i.ibb.co/fV1zdt44/Untitled.jpg) / [2](https://i.imgur.com/THLiMqL.jpeg)

- 根据库大小和用户界面重新排序了 [Light Novel Sites](https://fmhy.net/reading#light-novels),添加了标签,删除了小图书馆,并主演了两个突出的图书 (NovelCool + WuxiaClick). [Before vs After](https://i.ibb.co/DSRfsrM/Untitled.png) / [2](https://i.imgur.com/IMRUQoo.png)

- 删除了 [Audio Recommendation](https://fmhy.net/audio#curated-recommendations)部分 + 增加了新的部分为 [Song / Artist Discovery](https://fmhy.net/audio#song-artist-discovery). [Before vs After](https://i.ibb.co/zTDNZgr9/Untitled.png) / [2](https://i.imgur.com/V6h0mQx.png)

- 添加了两个标签 [Linux Cheat Sheets](https://fmhy.net/linux-macos#cli-cheat-sheets) + [Linux Distros](https://fmhy.net/linux-macos#linux-distros),谢谢了 [helxop](https://github.com/fmhy/edit/commit/9a9cd6dd47027ac63370b453ec86a943cdc0b9d6). [Before vs After](https://ibb.co/gMD8mKbw) / [2](https://i.imgur.com/hqqLsCB.png)

- 为了改善它的外观,我们改造了我们的保护.

- 添加了我们_PROTECTED_28__的备份/前端,并将我们的 [Backup](https://fmhy.net/other/backups) + [FAQ](https://fmhy.net/other/FAQ)页面移动到我们的网站,而不是 reddit/github.

- 改善了我们的[theme handler](https://github.com/fmhy/edit/pull/4386),这将帮助我们更容易地更换或添加未来的主题 (如万圣节,圣诞节等).

- 添加了360总安全性.如果您安装了任何应用程序,它将继续给用户设置"工具箱",或者如果您使用清洁,维修或优化选项,将在未经同意的情况下安装它.安装后,工具箱将然后 [modify your default apps](https://en.wikipedia.org/wiki/Criticism_of_Qihoo_360#Malicious_promotion) (如您的浏览器) 并将它们全部转换为360选项.

# 星星加入

- 视频下载中已有星级 [DDLBase](https://fmhy.net/video#download-sites),具有4K+1080p内容,仅使用快速主机,拥有巨大的图书馆.

- 在动漫流媒体中演出. 很好的来源,很好的UI,自动接下来,自定义播放器,多语言.

- 支持许多网站,质量很好,使用很容易.

- 游戏下载,大型库,快速主机,预安装,允许请求.

- 果网站中着着的 [Comix](https://fmhy.net/reading#manga),大图书馆,允许上传,用户界面很好,

- 无需注册,多个API集成,最后.fm集成,可定制,大图书馆.

- 它们有很好的定制播放器,很多功能,快速流动和令人印象深刻的UI.

- 现场体育中已有了主角,拥有快速的服务器和强大的UI.

- 通过Twitter/X定制,增加了很多额外的功能,可以隐藏你不使用的部分,改善用户界面,等等.

- 文件主机,速度很好,每文件25GB,最后下载后持续15天. 

- 星座在图像代码中,有无限的图像4,Qwen和纳米香.

- 在iOS流程中明星的 [Nuvio](https://fmhy.net/mobile#ios-streaming),它使用Stremio插件,并且比目前在iOS上可用的"Lite"版本有更少的限制.

- 在Android Torrenting中,已明星化 [Flud](https://fmhy.net/mobile#android-torrenting),允许绑定,保持良好,通过DNS可以轻松阻止广告.

- 据悉,目前已有很好的收录,由丹编辑.

- 演出了"重建/复兴"中的"保护48号"

- 饰演了Remakes/Recreations,拥有丰富的吉他英雄克隆,有数千首歌+播放列表,没有延迟,与所有吉他控制器一起工作,并支持MIDI鼓套件.

- 星在多人浏览器游戏中,有像断,地理游戏,卡片游戏等等品质版本.

- 在虚拟机中明星的 [QEMU](https://fmhy.net/system-tools#virtual-machines),在非窗户系统上表现良好,最适用于Virt-Manager或Vagrantup.

- 备份文件中明星 [Restic](https://fmhy.net/file-tools#file-backup),易于使用,多平台,在基准中表现良好.

- 编码AI中明了.
