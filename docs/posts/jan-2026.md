# 维基更新

- 在我们的备份页面添加了**[3个新实例](https://fmhy.net/other/backups)**。(Samidy, JBugel, ArtistGrid.)

- 在我们的颜色选择器中添加了**[Catppuccin](https://i.ibb.co/Ps8vDJL0/image.png)** / [2](https://i.imgur.com/558l4gH.png)选项。感谢Samidy完成这项工作。

- 添加了[教科书](https://fmhy.net/reading#textbooks)的新部分。

- 添加了[文件信息/元数据](https://fmhy.net/file-tools#file-info-metadata)的新部分。

- 将Linux + Mac种子客户端合并到[主部分](https://fmhy.net/torrenting#torrent-clients)，并添加了标签和平台图标。[前后对比](https://i.ibb.co/fV1zdt44/Untitled.jpg) / [2](https://i.imgur.com/THLiMqL.jpeg)

- 根据图书馆规模和UI重新排序了[轻小说网站](https://fmhy.net/reading#light-novels)，添加了标签，删除了小型图书馆，并推荐了两个突出的网站(NovelCool + WuxiaClick)。[前后对比](https://i.ibb.co/DSRfsrM/Untitled.png) / [2](https://i.imgur.com/IMRUQoo.png)

- 精简了[音频推荐](https://fmhy.net/audio#curated-recommendations)部分+添加了[歌曲/艺术家发现](https://fmhy.net/audio#song-artist-discovery)的新部分。[前后对比](https://i.ibb.co/zTDNZgr9/Untitled.png) / [2](https://i.imgur.com/V6h0mQx.png)

- 为我们的[评分页面](https://fmhy-grading.pages.dev/)添加了备份/前端，并将我们的[备份](https://fmhy.net/other/backups)和[常见问题](https://fmhy.net/other/FAQ)页面从reddit/github移到了我们的网站。感谢Roi Goat + Samidy完成这项工作。

- 改进了我们的[主题处理程序](https://github.com/fmhy/edit/pull/4386)，这将帮助我们在未来更容易地切换或添加主题(如万圣节、圣诞节等)。感谢Land完成这项工作。

# 新增推荐 ⭐

- 在视频下载中推荐了[DDLBase](https://fmhy.net/video#download-sites)，拥有4K + 1080p内容，只使用快速主机，拥有庞大图书馆。

- 在动漫流媒体中推荐了[Anime Realms](https://fmhy.net/video#anime-streaming)。优质资源，界面美观，自动下一集，自定义播放器，多语言。

- 在Telegram音频机器人中推荐了[Music Hunters](https://fmhy.net/audio#telegram-bots)。支持多个网站，质量良好，易于使用。

- 在游戏下载中推荐了[AstralGames ](https://fmhy.net/gaming#download-games)，图书馆庞大，主机快速，预安装，允许请求。

- 在漫画网站中推荐了[Comix](https://fmhy.net/reading#manga)，图书馆庞大，允许上传，界面美观，很快将添加更多语言。

- 在音频流媒体中推荐了[Monochrome](https://fmhy.net/audio#streaming-sites)，无需注册，多API集成，last.fm集成，可自定义，图书馆庞大。

- 在流媒体中推荐了[Rive, Aether, Willow, and FlyX](https://fmhy.net/video)。这些拥有美观的自定义播放器，功能丰富，流媒体快速，界面令人印象深刻。

- 在直播体育中推荐了[StreamSports99](https://fmhy.net/video#live-sports)，拥有快速服务器和稳定的界面。

- 在Twitter/X定制中推荐了[Control Panel for Twitter](https://fmhy.net/social-media-tools#twitter-x-customization) Twitter/X定制，添加了许多额外功能，让您隐藏不使用的部分，改进界面，等等。

- 在文件托管中推荐了[Rootz](https://fmhy.net/file-tools#file-hosts)，速度良好，每个文件25GB，最后一次下载后保存15天。

- 在图像生成中推荐了[PigenAI](https://fmhy.net/ai#image-generation)，拥有无限的Imagen 4, Qwen和Nano Banana。

- 在iOS流媒体中推荐了[Nuvio](https://fmhy.net/mobile#ios-streaming)，使用Stremio插件，比目前iOS上可用的"Lite"版本限制更少。

- 在Android种子下载中推荐了[Flud](https://fmhy.net/mobile#android-torrenting)，允许绑定，维护良好，可通过DNS轻松屏蔽广告。

- 在身体健康中推荐了[Sleeping Guide](https://fmhy.net/misc#physical-health)，睡眠卫生/科学指南，到目前为止似乎反响良好，由Dan策划。

- 在重制/重现中推荐了[OpenMW](https://fmhy.net/gaming#remakes-recreations)，完全从头开始重新实现Morrowind，[获得了Bethesda本人的批准](https://wiki.openmw.org/index.php?title=Bethesda_Emails)。

- 在重制/重现中推荐了[Clone Hero](https://fmhy.net/gaming#remakes-recreations)，功能丰富的Guitar Hero克隆，数千首歌曲+播放列表，无延迟，兼容所有吉他控制器，支持midi鼓套件。

- 在多人浏览器游戏中推荐了[Gidd](https://fmhy.net/gaming#party-multiplayer)，拥有高质量的垄断、geoguessr、纸牌游戏等版本。

- 在虚拟机中推荐了[QEMU](https://fmhy.net/system-tools#virtual-machines)，在非Windows系统上表现良好，最好与Virt-Manager或Vagrantup一起使用。

- 在文件备份中推荐了[Restic](https://fmhy.net/file-tools#file-backup)，易于使用，多平台，基准测试表现良好。

- 在编码AI中推荐了[Gemini CLI](https://fmhy.net/ai#coding-ais)，这似乎是目前限制最好的之一。

# 已移除项目

- 移除了CrocDB，因为它们已被[DMCA打击](https://i.ibb.co/RpDvJFgf/image.png) / [2](https://i.imgur.com/iSV0MHq.png)。

- 移除了IcebergCharts网站，因为它已经被[黑客攻击了好几次](https://i.ibb.co/XxMMwqDk/image.png) / [2](https://i.imgur.com/MkvElvr.png)，并且无法弄清楚为什么会发生这种情况。如果他们解决了这个问题，我们可能会在未来重新添加。请注意，我们现在会保留subreddit。

- 移除了CanvasBlocker，因为它是Firefox专用的，而FF现在已经内置了此功能，在某些情况下它实际上会使指纹识别变得更糟，并且不再很好地维护。

- 从直播体育中移除了FTV和WAC，因为两者现在已经不存在了。

- 移除了Foxified，因为它突然[用广告填充结果](https://i.ibb.co/4RzPsQ8Z/image.png) / [2](https://i.imgur.com/AjBMdfX.png)。

- 取消推荐了OnTheSpot，因为它很快将失去Spotify支持。

- 取消推荐了编码AI中的Windsurf，因为它的限制和免费模型不再那么好了。
