# 维基更新

- 制作了FMHY [Redlib实例](https://redlib.fmhy.net/r/FREEMEDIAHECKYEAH/wiki/index) + 我们的[SearXNG](https://searx.fmhy.net/)也重新工作了。

- 在游戏工具中添加了[ARPG / MMORPG](https://fmhy.net/gaming-tools#arpg-mmorpg-tools)部分。

- 在阅读中添加了[STEM](https://fmhy.net/reading#stem-resources) + [Kindle工具](https://fmhy.net/reading#kindle-tools)部分。

- 在视频工具中添加了[光盘工具](https://fmhy.net/video-tools#disc-utilities) + [视频增强/效果](https://fmhy.net/video-tools#video-enhancement-effects)部分。

- 将Steam / Epic部分分成子部分：[客户端工具](https://fmhy.net/gaming-tools#client-tools)，[DLC解锁/DRM绕过](https://fmhy.net/gaming-tools#dlc-unlock-drm-bypass)，[Steam商店](https://fmhy.net/gaming-tools#steam-store)。

- 将macOS部分分成子部分：[视频](https://fmhy.net/linux-macos#mac-video)，[音频](https://fmhy.net/linux-macos#mac-audio)，[图像](https://fmhy.net/linux-macos#mac-images)，[互联网](https://fmhy.net/linux-macos#mac-internet)，[系统](https://fmhy.net/linux-macos#system-tools)，[文件](https://fmhy.net/linux-macos#file-tools-1)。

- 将PDF工具分成子部分：[在线工具包](https://fmhy.net/file-tools#online-pdf-toolkits)，[离线工具包](https://fmhy.net/file-tools#offline-pdf-toolkits)，[转换](https://fmhy.net/file-tools#pdf-conversion-tools)，[实用工具](https://fmhy.net/file-tools#pdf-utilities)。

- 将视频处理分成[程序](https://fmhy.net/video-tools#processing-encoding)和[在线](https://fmhy.net/video-tools#online-processing)部分。

- 重新组织并为[安卓文本编辑器](https://fmhy.net/storage#android-text-editors)，[笔记](https://fmhy.net/storage#note-taking)和[待办事项](https://fmhy.net/storage#to-do-apps)添加标签。[前后对比](https://i.imgur.com/JyGeAsi.jpeg)

- 重新组织并为[环境音混合器](https://fmhy.net/storage#ambient-sound-mixers)和[安卓浏览器](https://fmhy.net/storage#android-browsers)添加标签。[前后对比](https://i.imgur.com/F48u9pk.png)

- 重新组织并为[网站模板](https://fmhy.net/storage#website-templates)，[Wordpress主题](https://fmhy.net/storage#wordpress-themes)和[YouTube电影](https://fmhy.net/video#free-w-ads)添加标签。[前后对比](https://i.imgur.com/Q84jTaz.png)

- 根据[投票结果](https://challonge.com/lzbv9baj/)重新组织[漫画部分](https://fmhy.net/reading#manga)。恭喜ComicK拔得头筹。

- 将[网站生成器](https://fmhy.net/developer-tools#website-generators)从存储中移出，重新组织，并添加标签。[前后对比](https://i.imgur.com/V1DWN9D.jpeg)

- 将[3D建模+CAD部分](https://fmhy.net/image-tools#_3d-models)从存储中移出，重新组织，并添加标签。[前后对比](https://i.imgur.com/4V41Zdx.png)

- 所有的媒体追踪部分现在都已分成追踪/数据库和推荐。这适用于流媒体、游戏、阅读和音频部分。

# 新增推荐 ⭐

- 在动漫流媒体中推荐了[AnimeKai](https://fmhy.net/video)。功能丰富，图书馆庞大，服务器快速。人们似乎真的很喜欢它，所以我们把它移到了第1位。

- 在驱动器/目录中推荐了[111477 + DramaSuki](https://fmhy.net/video#drives-directories)。两者都有快速的速度和大量内容。

- 在漫画网站中推荐了[BatCave](https://fmhy.net/reading#comics)。新的漫画网站，具有追踪、评分和美观的界面。

- 在P2P文件传输中推荐了[PairDrop](https://fmhy.net/file-tools#p2p-transfer)。开源，易于使用，允许创建房间。

- 在书籍推荐中推荐了[Five Books](https://fmhy.net/reading#curated-recommendations)。专家推荐的前5名列表，涵盖多个类别。

- 在游戏推荐中推荐了[VideoGameCanon](https://fmhy.net/gaming-tools#curated-recommendations)。游戏前1000名列表，我们服务器中的人觉得很有用。

- 在URL工具中推荐了[Mini QR](https://fmhy.net/internet-tools#url-tools)。让您创建自定义QR码艺术。

- 在安卓生产力追踪器中推荐了[Table Habit](https://fmhy.net/mobile#productivity-trackers)。开源，定期更新，具有美观的界面。

- 在背景移除器中推荐了[BGBye + pixelcut](https://fmhy.net/storage#background-removers)，因为它们似乎效果最好。

- 在百科全书中推荐了[EncycloSearch](https://encyclosearch.org/)。将几十个百科全书结合到一个搜索引擎中。

- 在音频提取中推荐了[Exact Audio Copy](https://fmhy.net/audio#audio-ripping-tools)。流行的CD提取工具，在Redacted的面试指南中被推荐。

- 切换到[AI Studio](https://fmhy.net/ai#online-chatbots)作为Google的主要聊天机器人，因为它在移动设备上工作良好，延迟较低，具有聊天历史记录，且比Gemini的审查较少。

- 添加了[Free-Official-Youtube-Content](https://fmhy.net/video#free-w-ads)作为免费广告内容中的索引表情符号。不是推荐，但我认为足够好值得一提。
