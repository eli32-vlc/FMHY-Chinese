# 维基更新

- 由Dan编写的全部指南，如**[/coffee/](https://rentry.co/coffeeguide)**和**[/pool/](https://rentry.co/poolguide)**，已汇编在**[/danfmhy/](https://rentry.co/danfmhy/)**中。

- 您现在可以使用[Brave Goggles](https://github.com/fmhy/bookmarks?tab=readme-ov-file#goggle)搜索FMHY。

- 在ROM中添加了[Nintendo](https://fmhy.net/gaming#nintendo-roms)、[Sony](https://fmhy.net/gaming#sony-roms)和[资源](https://fmhy.net/gaming#rom-resources)部分。

- 在益智游戏中添加了[魔方](https://fmhy.net/gaming#rubiks-cube)、[扫雷](https://fmhy.net/gaming#minesweeper)和[填字游戏](https://fmhy.net/gaming#crossword-puzzles)部分。

- 在教育中添加了[游戏学习](https://fmhy.net/educational#game-learning)部分，包含[魔方](https://fmhy.net/educational#rubiks-cube)和[国际象棋](https://fmhy.net/educational#chess)子部分。

- 在互联网中添加了[Firefox工具](https://fmhy.net/internet-tools#firefox-tools)部分。

- 在隐私中添加了[数据泄露监控](https://fmhy.net/privacy#data-breach-monitoring)部分。

- 在系统工具中添加了[任务自动化](https://fmhy.net/system-tools#task-automation)部分。

- 在Linux底部添加了[类Unix](https://fmhy.net/linux-macos#unix-like)部分。

- 在[远程种子](https://fmhy.net/torrenting#remote-torrenting)描述中添加了大小限制和注册要求。

- 根据[投票结果](https://i.imgur.com/2FOFOth.png)重新组织了[安卓音频播放器](https://fmhy.net/mobile#android-audio-players)。四个推荐现在是：Poweramp、Musicolet、VLC和Foobar。

- 根据[投票结果](https://i.imgur.com/OE8zsC0.png)重新组织了[安卓键盘](https://fmhy.net/mobile#keyboard-tools)。FUTO设法保持了其榜首位置，Heli排名大幅提升。

- 重新组织了[直播体育](https://fmhy.net/video#live-sports)，移除了损坏或重复播放器的网站，并推荐了一些似乎突出的网站。[前后对比](https://i.imgur.com/eDioF9L.png)。

- 重新组织了[电台流媒体](https://fmhy.net/audio#radio-streaming)和[网络电台](https://fmhy.net/audio#internet-radio)，添加了新推荐，新标签，并将两个部分从存储中移出。

- 重新组织了[RSS阅读器](https://fmhy.net/internet-tools#rss-readers)，改进了标签，移除了不良的。[前后对比](https://i.imgur.com/oAVEC6H.png)。

- 重新组织了[文字冒险](https://fmhy.net/gaming#text-adventures)，添加了标签，并从存储中移出。[前后对比](https://i.imgur.com/JrdEDC3.png)。

- 重新组织了开发工具网站，从存储中移出，并分成[索引](https://fmhy.net/developer-tools#dev-tool-indexes)和[在线工具包](https://fmhy.net/developer-tools#online-toolkits)。[前后对比](https://i.imgur.com/JyJOW9v.png)。

- 更新了[评分标准#4和#5](https://github.com/fmhy/FMHY/wiki/Stream-Site-Grading)以奖励在所有服务器上都有源自动切换+剧集自动下一集功能的网站。还将质量、字幕和媒体类型标准合并到#1中，以简化评分，并开始为观看派对支持给予积分。这减少了推荐总数，帮助最佳网站更加突出。[前后对比](https://i.imgur.com/omYr1pQ.png)。

# 新增推荐 ⭐

- 在图像生成中推荐了[LMArena](https://fmhy.net/ai#image-generation)。好模型，无限制，无需注册，如果遇到错误请清除cookie。

- 在在线聊天机器人中推荐了[HuggingChat](https://fmhy.net/ai#online-chatbots)。注重隐私，很多模型，需要注册，但临时邮箱可用。

- 在文件传输中推荐了[Blip](https://fmhy.net/file-tools#file-transfer)。轻松在互联网或您自己的设备之间传输大文件。我们discord中尝试过的每个人都很喜欢。

- 在安卓图像编辑器中推荐了[ImageToolbox](https://fmhy.net/mobile#camera-tools)。开源，功能丰富，维护良好。

- 在YouTube下载器中推荐了[Parabolic](https://fmhy.net/social-media-tools#youtube-downloaders)。开源，多平台，易于使用的YT-DLP前端之一。

- 在游戏追踪/数据库中推荐了[Infinite Backlog](https://fmhy.net/gaming-tools#tracking-databases)。Backlogged/Glitchwave竞争对手，具有美观的界面、成就集成和挑战。

- 在游戏模组中推荐了[GameBanana](https://fmhy.net/gaming-tools#game-mods)。高质量，组织良好，是某些系列如Sonic、Celeste等的首选。

- 在多人浏览器游戏中推荐了[TETR.IO](https://fmhy.net/gaming#party-multiplayer)。F2P多人俄罗斯方块，受欢迎，功能丰富，由其开发者[维护良好](https://tetr.io/about/patchnotes/)。

- 在浏览器游戏中推荐了[Mario Kart PC](https://fmhy.net/gaming#browser-games)。SNES风格马里奥赛车，具有[自定义地图](https://mkpc.malahieude.net/creations.php)和多人游戏。

- 在安卓音频均衡器中推荐了[RootlessJamesDSP](https://fmhy.net/mobile#android-audio)。开源，支持许多应用，AutoEQ支持，好的Poweramp替代品。

- 在GeoGuessr游戏中推荐了[Plonk It](https://fmhy.net/gaming#geoguessr-games)。学习GeoGuessr的活跃社区。

- 在编码AI中推荐了[Zed](https://fmhy.net/ai#coding-ais)。协作编码AI，从[HackerNews](https://news.ycombinator.com/item?id=43959710)和我们的成员那里获得了积极关注。
