# 维基更新

- 所有由丹写的指南,如**[/coffee/](https://rentry.co/coffeeguide)**和 **[/pool/](https://rentry.co/poolguide)**,都已编译成**[/danfmhy/](https://rentry.co/danfmhy/)**. 

- 现在可以使用 [Brave Goggles](https://github.com/fmhy/bookmarks?tab=readme-ov-file#goggle)搜索 FMHY.

- 添加了[Nintendo](https://fmhy.net/gaming#nintendo-roms), [Sony](https://fmhy.net/gaming#sony-roms)和 [Resource](https://fmhy.net/gaming#rom-resources)部分到ROM中.

- 加入了_PROTECTED_10__, [Minesweeper](https://fmhy.net/gaming#minesweeper)和 [Crossword](https://fmhy.net/gaming#crossword-puzzles)部分.

- 增加了教育部分的[Game Learning](https://fmhy.net/educational#game-learning)部分,其中包括[Rubik's Cube](https://fmhy.net/educational#rubiks-cube) + [Chess](https://fmhy.net/educational#chess)子部分.

- 加入了[Firefox Tools](https://fmhy.net/internet-tools#firefox-tools)部分. 

- 添加了_PROTECTED_17__部分到隐私.

- 增加了系统工具的 [Task Automation](https://fmhy.net/system-tools#task-automation)部分.

- 在Linux的底部添加了 [Unix-Like](https://fmhy.net/linux-macos#unix-like)部分.

- 增加了尺寸限制+注册要求,

- 根据[Poll Results](https://i.imgur.com/2FOFOth.png)重新组织了 [Android Audio Players](https://fmhy.net/mobile#android-audio-players).现在的四颗星星是:Poweramp,Musicolet,VLC和Foobar.

- 根据PROTECTED_24__重新组织了.FUTO成功保持了顶级位置,而Heli被撞上了更高的位置.

- 改编了[Live Sports](https://fmhy.net/video#live-sports),删除了破产或欺骗的网站,并演出了一些似乎突出的角色.

- 重新组织了[Radio Streaming](https://fmhy.net/audio#radio-streaming) + [Internet Radio](https://fmhy.net/audio#internet-radio),增加了新的星星,新的标签,并将两个部分从存储中移除.

- 改编了标签,删除了坏标签.

- 改组了[Text Adventures](https://fmhy.net/gaming#text-adventures),添加了标签,然后搬出了仓库.

- 重新组织了 Dev Tool Sites,从存储中移动,并分为 [Indexes](https://fmhy.net/developer-tools#dev-tool-indexes) + [Online Toolkits](https://fmhy.net/developer-tools#online-toolkits). [Before vs. After](https://i.imgur.com/JyJOW9v.png).

- 更新了 [grading criteria #4 and #5](https://github.com/fmhy/FMHY/wiki/Stream-Site-Grading),以奖励所有服务器上的自动交换源 + 节目自动接下来网站. 并且将质量,字幕和媒体类型标准合并到#1,简化得分,并开始为观看党的支持提供分数. 这减少了总数的星星,并帮助最好的网站更突出. [Before vs. After]( https://i.imgur.com/omYr1pQ.png)
