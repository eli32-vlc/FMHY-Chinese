# 维基更新

- 为我们的网站添加了**[AMOLED主题](https://i.ibb.co/hx97zL3W/978676.jpg)** / [2](https://i.imgur.com/fMrnGmF.png)作为可以开启或关闭的[切换](https://i.ibb.co/pvkfg3hC/image.png) / [2](https://i.imgur.com/qF7exKw.png)。感谢@Land完成此项工作。

- 构建了**[外部搜索引擎](https://fmhy-search.dev.zenir.tech/)**，在大多数情况下应该比我们网站上内置的VitePress搜索效果更好。

- 添加了FMHY的**[新备份](https://fmhyclone.pages.dev/)**，每日同步，托管在GitLab上。它还有一个[原始markdown](https://fmhyapi.wispy.qzz.io/single-page)页面的备份。我们还添加了另一个[备份](https://a-fmhy.pages.dev/)，上面自动开启了上述主题。

- 空间部分变得有点难以导航(60+行)，所以我们将其分成[自己的头部部分](https://fmhy.net/educational#space)，有两个新子部分：[天文学](https://fmhy.net/educational#astronomy)和[航天器](https://fmhy.net/educational#spacecraft)。天文学将涵盖与宇宙中天体或现象相关的事物。航天器将涵盖火箭、发射和国际空间站等内容。头部部分将用于更一般的太空相关内容，如NASA网站、新闻等。

- 游戏中的重制/移植超过了80行，相当混乱，所以我们将其分成3个新部分以使其更易理解：[分解/移植](https://fmhy.net/gaming#decomps-ports)、[重制/重现](https://fmhy.net/gaming#remakes-recreations)和[复兴项目](https://fmhy.net/gaming#revival-projects)。还将特殊兴趣变成自己的头部类别以更好地组织目录。

- 根据我们Discord中的投票结果重新排序了[漫画网站](https://fmhy.net/reading#manga)。Weeb Central已被移至第1位，MangaFire+MangaNato都已被推荐。感谢所有参与投票并发表意见的人。[前后对比](https://i.ibb.co/j9Sn4hRR/image.png) / [2](https://i.imgur.com/u8zFZTX.png)。

- 清理了多个音频流媒体部分，并添加了新部分以更好地组织，包括：[专业](https://fmhy.net/audio#specialty-streaming)、[特定类型](https://fmhy.net/audio#genre-specific-streaming)、[广播目录](https://fmhy.net/audio#radio-directories)和[Lo-fi广播](https://fmhy.net/audio#lofi-radio)。[前后对比](https://github.com/fmhy/edit/pull/4128#issuecomment-3476036920)。感谢@AnarchyDR完成此项工作。

- 更新了[音频流媒体](https://fmhy.net/audio)目录以使其不那么杂乱且更易于导航。[前后对比](https://i.ibb.co/0yJbh03H/234243.jpg) / [2](https://i.imgur.com/fhgqKzb.png)。

- 清理了[浏览器模拟器](https://fmhy.net/gaming#browser-emulators)，修复了标签，移除了失效网站，提升了有多款或独特模拟器的网站，将仅作为EmulatorJS/NeptunJS前端的网站移至存储。[前后对比](https://i.ibb.co/LXdhcDFD/Untitled.png) / [2](https://i.imgur.com/W9x8jY4.png)。

- 我们从ProtonVPN中移除了"不支持种子"标签，因为您可以使用允许免费种子的OpenVPN配置设置。请注意它们有时会过期且必须重新生成。指南列在[Proton](https://fmhy.net/privacy#vpn)旁边。感谢Wispy和其他人找出这一点。

- 在游戏中添加了[俄罗斯方块](https://fmhy.net/gaming#tetris)部分。

- 修复了游戏优化中的丑陋格式。[前后对比](https://i.ibb.co/Vc99kJhh/image.png) / [2](https://i.imgur.com/HRPUwL3.png)。

# 新增推荐 ⭐

- 在下载管理器中推荐了[AB Download Manager](https://fmhy.net/file-tools#download-managers)。开源，快速，跨平台，具有可恢复下载，以及活跃的开发团队。

- 将[CrocDB](https://fmhy.net/gaming#rom-sites)在ROM网站中提升至新第1位，超越Myrient。Croc涵盖多个网站(包括myrient)，具有更好的界面，以及新的rom包功能。

- 在种子流媒体应用中推荐了[PlayTorrio](https://fmhy.net/video#torrent-apps)。新客户端，功能非常丰富。它具有与Stremio类似的附加组件，如Torrentio。它还具有Jackett集成，Debrid支持，以及如此多其他功能，我们无法在此列出全部。

- 在专业聊天机器人中推荐了[NotebookLM](https://fmhy.net/ai#specialized-chatbots)。文档聊天机器人+笔记。非常适合快速学习，具有良好的音频/视频概览，可以生成测验、闪卡等。

- 在动漫流媒体中推荐了[Anidap](https://fmhy.net/video#anime-streaming)。具有美观界面，使用既独特又快速的主机。

- 在直播电视/体育中推荐了[PlayTorrio IPTV](https://fmhy.net/video#live-tv)。快速流媒体，庞大图书馆，良好质量，是我们一段时间内收到的较好的直播电视网站之一。请注意Darkness TV是原始版本，PlayTorrio只是改进了其界面。

- 在Linux游戏中推荐了[Heroic Games Launcher](https://fmhy.net/linux-macos#linux-gaming)。用于Epic、GOG和prime游戏的Linux游戏启动器，现在比Lutris维护得更好。

- 在阅读推荐中推荐了[ComicBookRoundup](https://fmhy.net/reading#curated-recommendations)。漫画焦点的评论/评分聚合器，类似于Metacritic或烂番茄。

- 在ROM网站中推荐了[Ziperto](https://fmhy.net/gaming#rom-sites)。存在很长时间了，具有稳固的图书馆，并使用快速主机。

- 在直播体育中推荐了[Sportsbite](https://fmhy.net/video#live-sports)。聚合器，具有许多主机，美观界面，涵盖大多数大型赛事。
