# 维基更新

- 添加了**[Amoled Theme](https://i.ibb.co/hx97zL3W/978676.jpg)** / [2](https://i.imgur.com/fMrnGmF.png)作为一个可以启动或关闭的 [Toggle](https://i.ibb.co/pvkfg3hC/image.png) / [2](https://i.imgur.com/qF7exKw.png).

- 建立了一个**[External Search Engine](https://fmhy-search.dev.zenir.tech/)**, 在大多数情况下,它应该比我们网站上的维特普্রেস搜索更好.

- 添加了每日同步的FMHY的 **[New Backup](https://fmhyclone.pages.dev/)**,在 GitLab上托管.它还有一个备份的 [raw markdown](https://fmhyapi.wispy.qzz.io/single-page)页面.我们还添加了另一个 [backup](https://a-fmhy.pages.dev/)的网站,自动从上方开始主题.

- 太空部分已经变得有些难以导航 (60+ 线) 所以我们将它分为 [own head section](https://fmhy.net/educational#space),有2个新子部分: [Astronomy](https://fmhy.net/educational#astronomy)和 [Spacecraft](https://fmhy.net/educational#spacecraft).天文学将涵盖与天体或宇宙中的现象相关的东西.太空船将涵盖火箭,发射和ISS等东西.头部部分将用于更一般的太空相关的事情,如NASA网站,新闻等.

- 游戏中的重版/端口有80多行,而且很不组织,所以我们把它分成3个新部分,使其更容易理解: [Decomps / Ports](https://fmhy.net/gaming#decomps-ports), [Remakes / Recreations](https://fmhy.net/gaming#remakes-recreations)和 [Revival Projects](https://fmhy.net/gaming#revival-projects).

- 根据我们Discord的民意调查结果重新订购了 [Manga Sites](https://fmhy.net/reading#manga).Weeb Central已被移至第1位,MangaFire + MangaNato都被主演.感谢参与投票并发表意见的人. [Before vs After](https://i.ibb.co/j9Sn4hRR/image.png) / [2](https://i.imgur.com/u8zFZTX.png).

- 清理了多个音频流媒体部分,并添加了新的部分来帮助更好地组织它,包括: [Specialty](https://fmhy.net/audio#specialty-streaming), [Genre Specific](https://fmhy.net/audio#genre-specific-streaming), [Radio Directories](https://fmhy.net/audio#radio-directories), 和 [Lofi Radio](https://fmhy.net/audio#lofi-radio). [Before vs. After](https://github.com/fmhy/edit/pull/4128#issuecomment-3476036920).

- 更新了[Audio Streaming](https://fmhy.net/audio)内容表,使其不太杂乱,更容易导航. [Before vs After](https://i.ibb.co/0yJbh03H/234243.jpg) / [2](https://i.imgur.com/fhgqKzb.png).

- 清理了[Browser Emulators](https://fmhy.net/gaming#browser-emulators),固定了标签,删除了死去的网站,撞击了多个网站,或者更高的独特模拟器,并将只作为模拟器JS / 尼普敦JS前端移到存储中. [Before vs After](https://i.ibb.co/LXdhcDFD/Untitled.png) / [2](https://i.imgur.com/W9x8jY4.png).

- 我们从ProtonVPN中删除了"没有流"标签,因为你可以使用一个开放VPN配置设置它,允许你免费流.请注意它们确实过期,有时必须被再生. 导向在 [Proton](https://fmhy.net/privacy#vpn)旁边列出.

- 加入了游戏的[Tetris](https://fmhy.net/gaming#tetris)部分.

- 修复了游戏优化中的丑格式化.

# 星星加入

- 开源,快速,跨平台,可重启下载,并有一个活跃的开发团队. 

- 击了 [CrocDB](https://fmhy.net/gaming#rom-sites)到 ROM 网站中的新#1.Croc 覆盖多个网站 (包括 myrient),具有更好的UI,以及新的 rompack功能.

- 讯流媒体应用程序中明星 [PlayTorrio](https://fmhy.net/video#torrent-apps). 新客户端非常丰富功能. 它具有类似于Stremio的补充功能,如Torrentio. 它还具有Jackett集成,Debrid支持和其他许多功能,我们无法在这里列出它们.

- 专业聊天机器人. 文件聊天机器人 + 笔记. 适合快速学习,有良好的音频/视频概述,可以生成测试,闪卡等. 

- 在动漫流媒体中演出. 它有一个漂亮的UI,使用独特和快速的主机.

- 在直播电视/体育中. 快速播放,庞大的图书馆,质量很好,我们在一段时间里发送的最好的直播电视网站之一. 注意,黑暗电视是原始的,PlayTorrio只是改善了他们的UI. 

- 星在Linux游戏中. Linux游戏发射器为Epic,GOG和主要游戏,现在更好地维护Lutris.
