# 东西被移除

- 现在,我们已经删除了安纳迪乌斯,如果有人新人来维护它,

- 由于Firefox已经删除了ContextSearch.它似乎导致问题,在某些情况下,网站被请求淹没.我们不确定它是否有意图恶意,或者其开发人员犯错误,但不管它建议删除它,并尝试其他选项,如 [ContextSearch web-ext](https://fmhy.net/internet-tools#firefox-extensions).

- 文件管理器中没有明星. 虽然最初有很多关于此的言,但人们开始说它太迟缓太有用.

- 没有星星的格洛克,奎恩,魔术和流动1. 施内尔在[Image Gen](https://fmhy.net/ai#image-generation). 这些被认为是好的过去,但现在有更好的,更少的日期免费选择.
