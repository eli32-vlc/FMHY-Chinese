# 维基更新

- 添加了新的备份网站**[FMHY.bid](https://fmhy.bid/)**。应该比.net目前在更多地方不受阻止。

- 添加了[摄影/相机](https://fmhy.net/image-tools#photography-cameras)的新部分。

- 添加了[3D打印/打印机](https://fmhy.net/image-tools#_3d-printing)的新部分。

- 重新排序了[AI基准测试](https://fmhy.net/ai#ai-benchmarks)，推荐了Kaggle基准测试，移除了有日期测试的，并为[专业基准测试](https://fmhy.net/ai#specialized-benchmarks)添加了子部分。[前后对比](https://i.ibb.co/5xY36Pk5/Untitled.png) / [2](https://i.imgur.com/NuIHGhC.png)。

- 重新排序并清理了[托管工具](https://fmhy.net/developer-tools#hosting-tools)中的丑陋格式。[前后对比](https://i.ibb.co/21g23vSy/Untitled.png) / [2](https://i.imgur.com/G6sJPo1.png)。

- 将[浏览器起始页](https://fmhy.net/internet-tools#browser-startpages) + [自定义新标签页](https://fmhy.net/internet-tools#custom-new-tab-pages)从存储中移出到自己的部分。

- 恢复了FMHY SafeGuard的[网站](https://fmhy.github.io/FMHY-SafeGuard/)。

# 新增推荐 ⭐

- 在动漫流媒体中推荐了[AnimeX](https://fmhy.net/video#anime-streaming)。功能丰富，可靠的来源，庞大的图书馆，美观的界面，有论坛。

- 在图像生成中推荐了[Bing图像创建器](https://fmhy.net/ai#image-generation)。每日提供200张，具有良好的编辑功能，似乎是目前使用GPT图像1的最佳方式。

- 在安卓电视中推荐了[⁠TizenTube Cobalt](https://fmhy.net/video#android-tv)。无广告YouTube应用，经常更新，支持Sponsorblock，使用官方YouTube界面。

- 在直播体育中推荐了[Watch Footy](https://fmhy.net/video#live-sports)。可靠的来源，美观的界面，许多赛事。

- 在直播流媒体工具中推荐了[Streamer.bot](https://fmhy.net/video-tools#live-streaming)。[功能丰富](https://streamer.bot/features)的直播流管理器，兼容OBS、Streamlabs等。

- 在Roblox工具中推荐了[RoSeal](https://fmhy.net/gaming-tools#roblox-tools)。改进Roblox网站并添加近200个额外功能。

- 在安卓有声读物播放器中推荐了[Voice](https://fmhy.net/mobile#android-reading)。开源，多格式，有封面获取，以及美观的极简设计。

- 在移动生产力中推荐了[Loop习惯追踪器](https://fmhy.net/mobile#productivity-trackers)。开源，有固定通知，比大多数免费选项有更好地限制。

- 重新推荐了[CDRomance](https://fmhy.net/gaming#rom-sites)在ROM网站中，因为它重新出现了。请注意RetroGamingTalk账户将在CDRomance上有效。

# 已移除项目

- 移除了Anadius，因为它们已[决定退出](https://i.ibb.co/TD9kRCR3/image.png) / [2](https://i.imgur.com/2Xk8Jor.png)。如果有人新接手维护，它可能会在未来重新添加。

- 移除了ContextSearch，因为它已被Firefox移除。它似乎正在引起问题，在某些情况下向网站发送过多请求。我们不确定它是否故意恶意，或是其开发者的错误，但无论如何建议移除它，并尝试其他选项，如[ContextSearch web-ext](https://fmhy.net/internet-tools#firefox-extensions)。

- 取消推荐了[Files](https://fmhy.net/file-tools#file-managers)在文件管理器中。最初对此有很多炒作，但人们开始说它太卡顿/缓慢而不实用。

- 取消推荐了Grok、Qwen、Mage和Flux.1 Schnell在[图像生成](https://fmhy.net/ai#image-generation)中。这些在过去被认为不错，但此时有更好、不过时的免费选项。
