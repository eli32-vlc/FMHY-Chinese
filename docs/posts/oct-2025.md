# 维基更新

- KEEP3/KEEP重新组织,重新测试,更快的主机更高,更好地移动适合其他部分的,删除死或仅限高档,并固定不准确的标签.

- KEEP5/KEEP重新组织,删除过时网站,修复了错误的格式化.

- 现在我们已经重新设计了所有VPN+代理部分. 太多的内容可以被视为"前与后"的图像,

- 修改了KEEP8/KEEP的格式化,将其转化为自己的头部,并添加了新的子部份为KEEP9/KEEP + KEEP10/KEEP,以使导航更容易.

- 添加PWA (进步Web应用) 支持我们的网站.这意味着我们的网站可以像应用程序一样添加到您的主屏幕或桌面.它可以 (理论上) 允许浏览网站时更顺利的体验.对于移动,它可能会提示,或者您可以通过选择KEEP11/KEEP添加它.对于桌面,只需按住地址中的KEEP12/KEEP.

- 由于他们不能再使用KEEP13/KEEP. 我们也击落了它在Proton以下,因为Proton是无限的,尽管没有相同的流限制.

# 星星加入

- KEEP14/KEEP在iOS部分中.

- 标有 KEEP15/KEEP在软件网站. 定制的多站点软件搜索引擎,仅使用已在FMHY上列出的可信网站.

- KEEP16/KEEP在AI聊天机器人中. 良好的免费限制,在LMArena上获得最高评级的开源模型,很快将添加K2推理.

- KEEP17/KEEP在专业聊天机器人中.无限,显示所有来源,有良好的搜索选项.

- KEEP18/KEEP在Manhwa / 漫画中. 很大的库,没有压缩,很好的UI,可以切换SFW / NSFW.

- KEEP19/KEEP在漫画/轻小说追踪中. 有广泛的,详细的数据库,和漂亮的UI,似乎是很喜欢到目前为止.

- 文件主机中明星KEEP20/KEEP. 快速,无限大小,文件持续15天 (最后一次下载后).现在似乎被很多网站使用.

- KEEP21/KEEP在购物部分中. 详细的wiki记录影响消费者权利的做法. 由Louis Rossmann创立.

- KEEP22/KEEP重新演,因为他们已经恢复了大量已删除的内容.

- 恢复了星际火52的功能.
