# 新增推荐 ⭐

- 在歌曲识别中推荐了[Google Assistant](https://fmhy.net/audio#song-identification)。通过哼唱歌曲工作，在我们的测试中似乎比Shazam更准确。

- 在图像生成中推荐了[Qwen](https://fmhy.net/ai#image-generation)。无限制，无需注册，目前似乎几乎和4o一样好。

- 在Reddit工具中推荐了[Redlib](https://fmhy.net/social-media-tools#reddit-tools)。注重隐私的reddit前端，无需账户即可使用。

- 在打字课程中推荐了[keybr](https://fmhy.net/text-tools#typing-lessons)。自定义触摸打字练习，具有美观的界面，多人游戏，个人资料，主题等。

- 在安卓YouTube下载器中推荐了[Seal](https://fmhy.net/mobile#android-youtube-apps)。开源下载器，支持许多网站。

- 在安卓电池工具中推荐了[SaverTuner](https://fmhy.net/mobile#battery-tools)。电池节省应用，在我们的测试中许多人取得了良好效果。

- 在安卓社交媒体中推荐了[Xtra](https://fmhy.net/mobile#social-media-apps)。开源Twitch客户端，对人们来说似乎效果良好。

- 在直播体育中推荐了[FSTV](https://fmhy.net/video#live-sports)。具有快速流媒体+覆盖了相当数量的联赛。

- 在音频元数据中推荐了[MusicBrainz Picard](https://fmhy.net/audio#audio-metadata)。大目录，适合批量元数据自动化和排序。

- 在饮料部分推荐了[Beanconqueror](https://fmhy.net/misc#drinks)。开源，无广告，[功能丰富](https://i.ibb.co/1GkdXk4N/image.png)的酿造追踪应用。

- 在填字游戏中推荐了[Crosshare](https://fmhy.net/gaming#crosswords)。功能丰富，自定义界面，活跃用户群和每日新谜题。

- 在计算机科学中推荐了[CS50](https://fmhy.net/educational#computer-science)。备受尊敬的哈佛计算机科学课程。

# 已移除项目

- 移除了nsw2u、nswdl、ps4pkg和game-2u，因为它们已被FBI针对并[关闭](https://torrentfreak.com/fbi-seizes-gaming-piracy-domains-including-pre-release-target-nsw2u/)。

- 移除了InstallGentoo维基，因为它已被恶作剧者接管。目前应避免访问该网站(NSFL内容)，但如果情况得到处理，我们会考虑重新添加。

- 移除了HuggingChat，因为它已[关闭](https://x.com/julien_c/status/1940082259287069089)。请注意HuggingFace仍然有效，只有聊天功能已消失。

- 从YT下载器中移除了Cobalt，因为它[不再支持YouTube](https://x.com/justusecobalt/status/1943279590178230618)。请注意它仍然支持其他视频网站，所以我们将其保留在适合的其他部分。

- 移除了Masquerade Repacks，因为它们已[宣布停止](https://i.ibb.co/0pndbGNP/image.png)，请注意kaos本身仍然强劲。
