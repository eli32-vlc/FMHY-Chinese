# 维基更新

- 添加了一个**[指南](https://fmhy.net/other/selfhosting)**来设置和在本地运行您自己的FMHY实例。

- 添加了一个FMHY Safeguard的**[用户脚本](https://greasyfork.org/en/scripts/528660-fmhy-safelink-guard)**版本。

- 您现在可以使用侧边栏中的图标选择网站的[自定义文本颜色](https://i.imgur.com/kXNRPjM.mp4)+更新了反馈系统使其更加突出。

- 清理了戏剧流媒体部分，移除了失效/不良网站，并推荐了一些似乎效果最好的，[前后对比](https://i.imgur.com/E3QTrUn.png)。

- 清理了日语学习部分，请注意没有移除网站，只是移动了位置，[前后对比](https://i.imgur.com/wPboWjk.png)。


- 为[AI视频生成器](https://fmhy.net/ai#video-generation)添加了使用限制标签。

- 更好地组织了字体部分+分成[生成器](https://fmhy.net/text-tools#font-text-generators)和[自定义](https://fmhy.net/text-tools#font-customization)。

- 为我们的流媒体网站分级系统添加了一些[新标准](https://i.imgur.com/s7UGdIz.png)。

# 新增推荐 ⭐

- 在音频下载中推荐了[DAB Music Player](https://fmhy.net/audio#download-sites)。一键FLAC专辑下载，具有网页应用+适用于windows、mac和linux的桌面应用，即将推出安卓支持。

- 在安卓隐私浏览器中推荐了[IronFox](https://fmhy.net/storage#privacy-based)。基于Firefox的浏览器，专注于隐私/安全。功能丰富，由librewolf开发团队推荐，我们的社区似乎很喜欢它。

- 在Twitch广告拦截器中推荐了[AdGuardExtra](https://fmhy.net/social-media-tools#twitch-adblockers)。这似乎是现在阻止广告的最佳方式。

- 在YouTube视频下载器中推荐了[AMP4](https://fmhy.net/social-media-tools#youtube-downloaders)，无广告，支持播放列表和3小时长的视频。

- 在网络隐私中推荐了[JustDeleteMe](https://fmhy.net/privacy#web-privacy)。用于更轻松地从网络服务删除账户的链接目录。

- 在播客流媒体中推荐了[PocketCasts](https://fmhy.net/audio)，流行的iOS播客播放器，最近添加了桌面+网页应用。

- 在航班部分推荐了[ADS-B Exchange](https://fmhy.net/misc#flights)，是少数几个不在付费墙后面锁定功能的网站之一。

- 用[ReVanced Manager](https://fmhy.net/mobile#android-audio)替换xManager的推荐，作为无广告Spotify，因为它具有更多功能，更少问题，xManager本身也推荐它。

- 移除了bark的推荐，因为它是有限的，以及Tortoise TTS的推荐，因为它听起来不好，而是推荐了[TTS Online](https://fmhy.net/ai#text-to-speech)，听起来更好，具有每日10k字符限制。

# 已移除项目

- 移除了TorrentGalaxy，看起来它不会很快回来。

- 从音频下载中移除了The Last Disaster，因为它们已经关闭。

- 从直播电视中移除了TOTV，因为每个频道现在似乎都是YouTube视频。

- 取消推荐了Plex，因为它们将远程流媒体[改为仅付费](https://www.plex.tv/blog/important-2025-plex-updates/)，同时提高了价格。

- 取消推荐了PSARips，因为它们的下载过程非常烦人。

- 取消推荐了Bookracy，因为它已经出现问题一段时间了。
