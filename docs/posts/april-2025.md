# 维基更新

- 添加了一个 **[Guide](https://fmhy.net/other/selfhosting)** 设置和运行您自己的本地FMHY实例.

- 增加了FMHY保护的**[Userscript](https://greasyfork.org/en/scripts/528660-fmhy-safelink-guard)**版本.

- 现在可以选择_PROTECTED_5__,使用侧边图标,

- 清理了剧情流媒体部分,删除了死/坏网站,

- 清理了日本学习部分,请注意没有网站被移除,只是移动,


- 添加了使用限制标签到 [AI Video Generators](https://fmhy.net/ai#video-generation).

- 更多的字体部分+分为 [Generators](https://fmhy.net/text-tools#font-text-generators)和 [Customization](https://fmhy.net/text-tools#font-customization).

- 增加了一些[New Criteria](https://i.imgur.com/s7UGdIz.png)到我们的流媒体网站评级系统.

# 星星加入

- 单击FLAC专辑下载,有网页应用程序 + 桌面应用程序用于Windows,Mac和Linux,Android支持即将到来.

- 基于Firefox的浏览器,重点是隐私/安全.功能丰富,由自由狼开发团队推,我们的社区似乎非常喜欢它.

- 现在,这似乎是阻止它们的最佳方式.

- 无广告,支持播放列表和3小时长的视频.

- 链接目录,可以更轻松地删除您的帐户从网页服务.

- 星在播客流. 热门的iOS播客播放器最近添加了桌面+网络应用程序.

- 航班部分中,只有一个没有锁定付费墙的功能.

- 换了xManager的星星在广告免费Spotify中用 [ReVanced Manager](https://fmhy.net/mobile#android-audio),因为它有更多功能,问题减少, xManager本身推它.

- 由于它听起来不好,所以它取消了 bark 的星球,并且取消了 Tortoise TTS 的星球,而其反之则是 [TTS Online](https://fmhy.net/ai#text-to-speech),这听起来更好,并且每天有10k 字符的限制.
